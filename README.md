# SocialApp

## Instalatie instructies:

### API-server
- cd GRAPHQL-api-server
- npm i
- npm start

### frontend
- install flutter cli
- cd social_app
- flutter build web
- folder met de build kan je vinden in: social_app/build/web
- deze folder kan gerunt worden op eenderwelke webserver
